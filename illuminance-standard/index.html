提示：照度单位为 lx（勒克斯）；体育建筑按训练 / 业余比赛 / 专业比赛（或电视转播档）分档列出； 室外区域以 Eav（维持平均照度）、Emin（水平最小）、垂直 Emin、半柱面 Emin 表示； 博物馆对光敏感展品以照度限值（≤）与年曝光量表示。U₀ / UGR / GR / Rₐ 缺失表示该场所标准未做规定。
