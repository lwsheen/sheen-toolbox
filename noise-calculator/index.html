    分级详情

    GBZ/T 229.4-2012 噪声作业危害分级表
    危害等级分级名称LEX,8h (dB(A))管理措施
    Ⅰ级轻度危害85 ≤ LEX,8h < 90常规监测，发放护听器
    Ⅱ级中度危害90 ≤ LEX,8h < 94加强监测，强制佩戴护听器，工程控制
    Ⅲ级重度危害94 ≤ LEX,8h < 97限期治理，严格健康监护
    Ⅳ级极重危害LEX,8h ≥ 97立即停产整改，高风险管理
    注：1. LEX,8h < 85 dB(A) 不属于噪声作业，无需分级。
    2. 交换率 q=3 dB（ISO/中国标准），即声级每增加 3 dB，允许时间减半。
